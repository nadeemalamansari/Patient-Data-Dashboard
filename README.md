# Patient Data Dashboard

> Coalition Technologies - Skills Test Submission

---

## Overview

Single-page responsive web application displaying **Jessica Taylor's** patient data by consuming the Coalition Technologies Patient Data API. Converted from the provided Adobe XD template.

## Technologies Used

- HTML5
- CSS3 (Custom)
- Vanilla JavaScript
- Chart.js v4.4.0
- Font Awesome v6
- REST API

---

## Features

- Patient information display (name, ID, DOB, gender, diagnosis, weight, blood pressure)
- Interactive blood pressure trend chart (2020-2025)
- API integration with mock data fallback
- Fully responsive design
- Single HTML file
